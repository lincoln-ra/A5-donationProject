Important FAQ

What is Document Object Model (DOM)?

Dom is a programming interface which represents the structure of a webpage like a tree and allow JavaScript to access, manipulate and update the contents of the the webpage.

How do you select an element from DOM?

We can select and element from DOM by their

  • ID
  • ClassName
  • TagName
  • QuerySelector
and more.

What is event Delegation in the context of the DOM and why is it useful?

Event Delegation is a technique where we can attach a single event listener to a parent element and handle it's child elements.
It is useful because it

  • Gives better experience by improving efficiency
  • It works for element that is added later and don't even need to add a new listener.
and more.

How do you manipulate an element's attributes and style using DOM?

To manipulate an element's attributes and style using DOM by

  • Getting Attributes or style
  • Setting Attributes or style
  • Removing Attributes or style
